# Digital Miners Rarity System

## Rarity Attributes

Digital Miners have 12 customizable attributes that affect rarity:

| # | Attribute | Category | Description |
|---|-----------|----------|-------------|
| 1 | Basement | Structure | Base foundation component |
| 2 | Body | Structure | Main chassis design |
| 3 | Buttons | Interface | Control panel buttons |
| 4 | Bottom fan | Cooling | Lower cooling system |
| 5 | Top fan | Cooling | Upper cooling system |
| 6 | Items | Accessories | Additional items and accessories |
| 7 | Screen | Interface | Display screen design |
| 8 | Stickers | Cosmetic | Decorative stickers |
| 9 | UI | Interface | User interface design |
| 10 | Upgrades | Enhancement | Upgrade modules |
| 11 | Walls | Structure | Housing panel design |
| 12 | Down Fan | Cooling | Double fan option |

## Rarity Notes

- There is **no strict mathematical ratio** to the rarity values
- Rarity information is hosted on OpenSea for Ethereum and BSC collections
- Rarity affects collectible value but not mining power
- Each attribute has multiple possible variants
- Combinations create unique miner appearances

## Checking Rarity

1. **OpenSea** (Ethereum/BSC): View rarity rankings on collection page
2. **Magic Eden** (Solana): Check trait rarity in item details
3. **Getgems** (TON): View attributes in NFT metadata
