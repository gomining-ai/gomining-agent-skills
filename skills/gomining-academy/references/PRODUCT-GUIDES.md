## Simple Earn

**Article:** https://academy.gomining.com/articles/what-is-simple-earn

### Overview

Simple Earn is GoMining's yield-generating product that allows users to earn returns on their crypto holdings.

### Features

| Feature | Description |
|---------|-------------|
| Supported Assets | BTC, GOMINING, USDT, USDC |
| Payout Frequency | Daily |
| Lock-up Period | None required |
| Flexibility | Full withdrawal flexibility |

### How It Works

```
1. Deposit supported crypto assets
       │
       ▼
2. Assets generate yield automatically
       │
       ▼
3. Receive daily BTC payouts
       │
       ▼
4. Withdraw anytime (no lock-up)
```

### Benefits

- Passive income generation
- Multiple asset support
- No lock-up requirements
- Daily reward distribution
- Flexible withdrawals

## Miner Wars

**Article:** https://academy.gomining.com/articles/what-is-miner-wars

### Overview

Miner Wars is GoMining's competitive game mode where users earn BTC rewards through teamwork and strategy.

### Features

| Feature | Description |
|---------|-------------|
| Reward Type | BTC |
| Gameplay | Strategy-based competition |
| Social | Clan system |
| Mode | Alternative to passive mining |

### Game Mechanics

| Element | Description |
|---------|-------------|
| Clans | Join or create teams |
| Competition | Compete against other players |
| Strategy | Use tactics to maximize rewards |
| BTC Collection | Earn Bitcoin through gameplay |

### How to Participate

1. Switch from Mining Mode to Miner Wars Mode
2. Join or create a clan
3. Compete in challenges
4. Collect BTC rewards

### GOMINING Token Integration

- Use tokens to activate boosts
- Enhance performance in competitions
- Unlock special abilities

## VeGOMINING (Vote-Escrowed)

**Course Lesson:** GoMining Basics, Lesson 8

### Overview

VeGOMINING is the governance token mechanism that allows GOMINING holders to participate in ecosystem decisions.

### Features

| Feature | Description |
|---------|-------------|
| Governance | Vote on proposals |
| Rewards | Enhanced earning rates |
| Influence | Shape ecosystem development |
| Lock-up | Time-weighted voting power |

### How It Works

1. Lock GOMINING tokens
2. Receive veGOMINING (vote-escrowed)
3. Voting power based on lock duration
4. Participate in governance decisions
5. Earn enhanced rewards
