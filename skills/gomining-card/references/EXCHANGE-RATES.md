# GoMining Card Exchange Rates

## Conversion Overview

The GoMining Card performs instant cryptocurrency-to-fiat conversion at the time of each transaction.

## How It Works

```
1. User makes purchase with card
2. Card selects crypto from wallet (GOMINING, BTC, USDT, or USDC)
3. Crypto converted to fiat at internal rate
4. Merchant receives fiat payment
5. Transaction completes instantly
```

## Exchange Rate Policy

| Aspect | Details |
|--------|---------|
| Rate Type | Internal exchange rate |
| Commission | No commission charged |
| Rate Source | GoMining internal pricing |
| Public Rates | May differ from public sources |

## Important Notes

> **Note:** There is no commission; however, GoMining uses an internal exchange rate, which might be different from publicly available sources.

### Rate Considerations

1. **Internal Rate** - GoMining determines the exchange rate
2. **No External Fees** - No additional commission on conversion
3. **Rate Variance** - Rates may differ from CoinMarketCap, CoinGecko, etc.
4. **Real-Time** - Rate applied at transaction time

## Supported Currencies

| Crypto | Fiat Conversion |
|--------|-----------------|
| GOMINING | Yes |
| BTC | Yes |
| USDT | Yes |
| USDC | Yes |

## Best Practices

1. Check wallet balance before large purchases
2. Consider rate differences for large transactions
3. USDT/USDC typically have more stable conversion rates
4. BTC and GOMINING rates may fluctuate more
