# Supported Regions for GoMining Card

## Eligibility Overview

| Region | Status | Notes |
|--------|--------|-------|
| European Economic Area (EEA) | Supported | Full access with EEA documents |
| United Kingdom | Supported | Full access |
| Switzerland | Supported | Full access |
| Other Europe | Varies | Check specific country |
| Americas | Limited | Restrictions apply |
| Asia-Pacific | Limited | Restrictions apply |
| Middle East | Limited | Restrictions apply |
| Africa | Limited | Restrictions apply |

## Special Requirements

### Belarus & Russia
| Country | Passport Accepted | Alternative |
|---------|-------------------|-------------|
| Belarus | No | Valid EEA+ residence permit required |
| Russia | No | Valid EEA+ residence permit required |

## Prohibited Jurisdictions

Card applications are **not available** in 60+ countries including but not limited to:

- Countries under international sanctions
- High-risk jurisdictions per FATF
- Countries with regulatory restrictions

> **Note:** The full list of prohibited jurisdictions is maintained by GoMining and may be updated. Check https://help.cards.gomining.com/ for the current list.

## Document Cross-Border Policy

| Scenario | Allowed |
|----------|---------|
| ID from Country A, Address from Country B | Yes |
| European ID, Non-European Address | Case by case |
| Non-European ID, European Address | Case by case |

## Verification Tips

1. Ensure documents are clear and readable
2. Check document expiration dates
3. Address documents must be under 3 months old
4. Names must match across all documents
