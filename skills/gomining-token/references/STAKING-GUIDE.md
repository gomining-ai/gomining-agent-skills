# GOMINING Token Staking Guide

## Overview

Staking GOMINING tokens allows holders to earn passive rewards while contributing to the ecosystem's security and governance.

## Staking Options

### Standard Staking

| Feature | Description |
|---------|-------------|
| Minimum Amount | Check current requirements on platform |
| Lock Period | Flexible options available |
| Rewards | GOMINING tokens + potential BTC |
| Withdrawal | Subject to unstaking period |

### VeGOMINING (Vote-Escrowed)

For enhanced benefits, users can lock GOMINING tokens to receive veGOMINING:

| Feature | Description |
|---------|-------------|
| Mechanism | Time-weighted locking |
| Voting Power | Proportional to lock duration |
| Rewards | Enhanced earning rates |
| Governance | Participate in ecosystem decisions |

## Reward Calculation

Staking rewards are influenced by:

| Factor | Impact |
|--------|--------|
| Amount Staked | More tokens = higher rewards |
| Lock Duration | Longer locks = bonus multiplier |
| Total Pool Size | Rewards distributed proportionally |
| Platform Activity | Ecosystem fees may increase rewards |

## VeGOMINING Details

### Lock Duration Multipliers

| Lock Period | Voting Power | Reward Bonus |
|-------------|--------------|--------------|
| Short-term | Lower | Base rate |
| Medium-term | Medium | Enhanced |
| Long-term | Maximum | Highest |

### Governance Rights

VeGOMINING holders can:
- Vote on protocol proposals
- Influence ecosystem development
- Shape reward distributions
- Participate in community decisions

## Unstaking

### Standard Staking
- Subject to cooldown period
- Rewards continue during unstaking
- Full token return after cooldown

### VeGOMINING
- Lock duration must complete
- Early exit may incur penalties
- Voting power decreases over time

## Best Practices

1. **Start Small** - Test with a smaller amount first
2. **Understand Locks** - Know the commitment before locking
3. **Monitor Rewards** - Track earnings regularly
4. **Participate in Governance** - Use voting rights actively
5. **Compound When Possible** - Reinvest rewards for growth
