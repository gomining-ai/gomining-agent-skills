# Mining Mode vs Miner Wars Mode

Each Digital Miner can run in **one of two modes** at a time. The setting is per-miner — you can split your fleet between modes if you want.

---

## Side-by-side Comparison

| Aspect | Mining Mode | Miner Wars Mode |
|--------|-------------|-----------------|
| **Activation** | Default after purchase | Switch in per-miner settings |
| **Participation** | None — fully passive | Active gameplay (or Spell Bot) |
| **Reward currency** | BTC | BTC (clan) + GOMINING (personal) |
| **Reward cadence** | Daily | Weekly (Tuesday 00:00 UTC) |
| **BTC source** | Mining pool payouts (Binance / VIA BTC / Foundry) | Clan victory pool (same pool rewards reallocated) |
| **GOMINING earning** | None (rewards are BTC) | Yes — personal battle prize fund |
| **Spend on the game** | None | GOMINING for spells/power-ups (optional) |
| **Skill / strategy** | None | Clan choice, spell timing, multiplier reading |
| **Reinvestment** | Available (TH or GOMINING auto-buy) | Available |
| **Ambassador rewards (referred)** | Calculated daily | Calculated weekly |
| **VIP maintenance discount** | Applies | Applies |
| **Risk profile** | Predictable, smooth | Variable — depends on Clan / personal performance |

---

## When to Stay in Mining Mode

* You want predictable daily BTC and no engagement
* Your miner is enrolled in a monthly power upgrade subscription, or is being sold, or was purchased via MNPL (these miners cannot reinvest into TH and don't suit active play)
* You don't want to manage spells / Clan membership

---

## When to Switch to Miner Wars Mode

* You're willing to spend GOMINING tokens on spells for higher expected returns
* You want to earn GOMINING (not just BTC)
* You're in a strong Clan (high PPS, high league)
* You want Clan ownership benefits (Gold I+ VIP, 5% royalties)

---

## Switching Effects

* Switch is done in the miner's mode setting
* Switching mid-Cycle: the miner contributes to whichever mode it's currently set to; rewards already earned in the previous mode are kept
* If you're excluded from a Clan as a non-owner, your miner **auto-switches back to Mining Mode**

---

## Hybrid Strategy

You can split your fleet:

* Keep high-power, high-efficiency miners in Miner Wars Mode (better PPS, better Clan contribution)
* Keep older or pre-MNPL miners in Mining Mode (passive BTC, eligible for reinvestment)

There is no platform-wide "all miners must be in one mode" rule.
