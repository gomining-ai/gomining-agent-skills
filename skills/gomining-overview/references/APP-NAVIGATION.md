# App Navigation — Section Paths

Use these section names when directing users to specific parts of the GoMining app. The system automatically converts section names into navigation links.

## My Miners

| Section | Path in App | When to Reference |
|---------|-------------|-------------------|
| **Mining Farm** | My miners → Mining Farm | Viewing all miners, miner status, Service button |
| **Dashboard** | My miners → Dashboard | Viewing mining statistics and performance |
| **Rewards (Solo)** | My miners → Rewards → Solo | Viewing or withdrawing BTC rewards in Mining mode |
| **Rewards (Miner Wars)** | My miners → Rewards → Miner Wars | Viewing Miner Wars reward history |
| **Selling** | My miners → Selling | Listing a miner for sale |

## Marketplace

| Section | Path in App | When to Reference |
|---------|-------------|-------------------|
| **Marketplace** | Marketplace → Miners | Buying miners from the secondary market |
| **GoMiners Collection** | Marketplace → GoMiners | Viewing and buying avatar NFTs |
| **Merch** | Marketplace → Merch | Purchasing GoMining branded merchandise |

## GOMINING Token
