# Raffle Ticket Eligibility

## What qualifies for raffle tickets?

| Action | Eligible? |
|--------|-----------|
| Miner upgrade (any source, including secondary market) | Yes — if within the active raffle cycle |
| New miner purchase from GoMining | Yes — if within the active raffle cycle |
| Secondary market miner purchase (no upgrade) | No — permanently excluded |
| Actions before raffle start | No |

## Key Rules

- **Cycle timing:** Eligibility is tied to specific raffle cycles. Each cycle starts at a defined time (e.g., 12:00 UTC). Upgrades or purchases from a prior cycle do not count toward the current raffle.
- **Secondary market miners:** Buying a miner on the secondary market never earns raffle tickets, even after the raffle starts. This is a permanent rule.
- **Upgrades apply to all miners:** Upgrading any miner — including a secondary market miner — is eligible, as long as the upgrade happens within the current raffle cycle.

## GoBox Raffle (Missing Ticket)

If you completed a qualifying TH upgrade but did not receive a raffle ticket:
- Click the **Take Part** button in the raffle section manually.
- Only **TH (hashrate) upgrades** qualify — energy efficiency (W/TH) upgrades do not.
- If you did a TH upgrade but the button is missing, contact support with a screenshot of the upgrade transaction.
