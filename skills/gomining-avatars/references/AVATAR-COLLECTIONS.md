# Avatar Collections Reference

## Overview

GoMining avatars are cosmetic items assigned to Digital Miners. There are four distinct races, each with unique visual aesthetics.

## Luminar VIP Boost Rules

| Rule | Detail |
|------|--------|
| Boost | +1 VIP level immediately upon assignment |
| TH Requirement | 500+ TH |
| Removal | VIP level reverts when avatar is removed |
| GoBox | Issued once for the new VIP level |
| Re-assignment | GoBox is NOT re-issued on reassignment |
| Bronze GoBox | NOT issued (anti-farming protection) |

## Assignment Rules

- Each miner can have one avatar assigned at a time.
- Avatars can be reassigned between miners.
- Trading an avatar on the marketplace removes it from the miner it was assigned to.
- Luminar VIP boost only applies while the avatar is actively assigned.
